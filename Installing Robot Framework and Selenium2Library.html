Introduction

These instructions cover installing Robot Framework and Selenium2Library that are needed in hands-on exercises. If you already have preconditions Python, Firefox and pip installed, you only need to run the following commands.

pip install robotframework
pip install robotframework-selenium2library

Installing Python

Python 2.6.x or 2.7.x is required by Selenium2Library. These instructions cover installing Python on Windows. On other operating systems you most likely already have suitable Python interpreter installed. If not, consult the documentation of your operating system for instructions.

  1. Download either 32bit or 64bit installer, from http://python.org/download/. If you are unsure which one to use, select the 32bit version.

  2. Run the installer. Default values are OK for all question.

  3. Add Python installation directory and the directory where Python scripts are installed into PATH. Assuming you have installed Python to C:\Python27, you need to add the following to the end of PATH environment variable:

    ;C:\Python27;C:\Python27\Scripts

    If you do not know how to modify PATH, see, for example, http://www.computerhope.com/issues/ch000549.htm.

Installing pip

pip is a tool for installing and managing Python packages. It makes installing, upgrading, and uninstalling Robot Framework and related Python based tools very convenient. pip is a replacement for easy_install which it also uses internally.

  1. Install easy_install by executing ez_setup.py file:

    python ez_setup.py

  1. Install pip by executing get-pip.py file: 

    python get-pip.py

Installing Robot Framework

Once you have pip installed, installing Robot Framework is very simple:

pip install robotframework

To later upgrade the framework or to get a specific version:

pip install --upgrade robotframework
pip install robotframework==2.7.6

Installing Selenium2Library

Installing Selenium2Library is as easy as installing Robot Framework:

pip install robotframework-selenium2library

Selenium2Library installation automatically installs various dependencies, including Selenium 2 tool it uses internally.

The browser we use in exercises is Firefox, so you should also have that installed.

Troubleshooting proxy problems

If pip complains that "Cannot fetch index base URL http://pypi.python.org/simple/" or other ways seems to have connection problems, you may need to add your proxy to your environment variables:

set HTTP_PROXY=10.144.1.10:8080

set HTTPS_PROXY=10.144.1.10:8080

You will then also need this environment variable to prevent connection problems with Selenium2Library:

set NO_PROXY=127.0.0.1,localhost
